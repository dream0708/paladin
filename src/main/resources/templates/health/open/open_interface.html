接口信息

健康处方获取接口

/open/diagnose

POST

application/json

JSON

获取健康处方

请求参数

参数名 参数类型 参数说明 值内容

返回参数

参数名 参数类型 参数说明
status int 返回状态（1成功、0系统异常、2调用失败）
message string 返回结果的说明，返回状态异常时可查看该参数
result DiagnosePrescription 返回结果，健康处方

DiagnosePrescription

参数名 参数类型 参数说明
diagnoseId string 该次健康诊断ID
factors List<PrescriptionFactor> 危险因素
prescriptions List<Prescription> 健康处方
terminologies List<PrescriptionTerminology> 说明、注释、附录

PrescriptionFactor

参数名 参数类型 参数说明
code string 危险因素代码
name string 危险因素名称
type string 危险因素类型（1：疾病2：危险因素3：非直接危险因素）
parentFactor string 父危险因素（如果多个用逗号[,]分隔）

Prescription

参数名 参数类型 参数说明
id string 健康处方ID
type int 健康处方类型（1：日常生活3：运动锻炼4：饮食习惯6：心理8：其他）
content string 健康处方内容
terminology string 健康处方内容中的术语（多个用逗号[,]分隔）

PrescriptionTerminology

参数名 参数类型 参数说明
name string 术语名称
nickName string 别名（多个用逗号[,]分隔）
type int 类型（1：文字2：页面3：文字+页面）
content string 文字内容
pageName string 页面名称
pageUrl string 页面URL,/static/html/appendix/{pageUrl}.html
